# 🎮 Jokenpô com Inteligência Artificial (C++)

Projeto desenvolvido em C++ que simula um jogo de pedra, papel e tesoura com uma IA adaptativa.

---

## 🧠 Diferencial

A IA analisa o padrão do jogador e tenta prever sua próxima jogada.

---

## ⚙️ Funcionalidades

- Entrada de jogadas do usuário
- IA com aprendizado baseado em histórico
- Sistema de decisão inteligente
- Aleatoriedade controlada

---

## 🔍 Como funciona

- Usa uma matriz 3x3 para registrar padrões de jogadas
- Analisa a última jogada do usuário
- Escolhe a melhor resposta baseada em probabilidade

---

## 🛠️ Tecnologias

- C++
- <random> (mt19937)
- Programação orientada a objetos

---

## ▶️ Como executar

```bash
g++ *.cpp -o jogo
./jogo
